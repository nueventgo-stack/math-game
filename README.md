✨ 遊戲特色
1. 無壓力學習環境
    * 無倒數計時：沒有時間壓力，孩子可以依照自己的步調思考。
    * 正向回饋：答對時會有「挖到寶石」的驚喜感；答錯時只有苦力怕溫和的搖頭，沒有刺耳的錯誤音效。
    * 具象化教學：內建「史蒂夫小教室」，用麥塊的物品（如方塊、蜘蛛腿）來具體解釋乘法概念，幫助理解。
2. 成就感驅動
    * 連續挖礦挑戰：畫面右上角的「火焰計數器」會記錄連續答對次數，激發孩子想「延續紀錄」的動力。
    * 大師里程碑：當孩子將某個數字（例如 7 的乘法）的所有題目都練完後，會獲得專屬的「乘法大師獎盃」，給予榮譽感。
3. 完全免費且免安裝
    * 這是一個單純的網頁檔案，不需要安裝任何軟體，不需要註冊帳號，甚至不需要網路也能玩！
🕹️ 遊玩機制說明
遊戲分為兩個主要模式：
1. 背誦合成表 (練習模式)
適合剛開始接觸乘法，或是對某個數字還不熟練的孩子。
* 玩法：選擇一個數字（例如 3），畫面會列出 3x1 到 3x9 的算式。
* 互動：答案一開始被石頭蓋住。孩子唸出答案後，點擊石頭「挖開」它，確認是否正確。
* 目標：把所有的石頭都挖開，象徵熟練該數字的乘法表。
2. 挑戰模式 (測驗模式)
適合已經背過乘法表，準備驗收成果的孩子。
* 玩法：
    * 指定數字挑戰：選擇特定數字（如 8），系統會隨機出題，但保證會把 8x1 到 8x9 全部出完一遍，不重複。全部答對即可解鎖「大師成就」。
    * 隨機挑戰：適合總複習，題目完全隨機。
* 提示功能：如果卡關了，可以點擊題目旁邊的「✨ 星星按鈕」，史蒂夫會跳出來給予提示（例如：「想像有 8 隻蜘蛛...」）。
💻 如何在教室/安親班使用？
這個遊戲最方便的地方在於**「離線使用」**。您不需要擔心學校網路不穩，也不用怕孩子趁機上網看影片。
步驟 1：取得檔案
1. 請下載或複製我提供的 MinecraftMath.html 檔案。
2. 您可以將這個檔案放在：
    * 教室電腦的桌面。
    * USB 隨身碟中，隨插隨用。
    * 學校的共享資料夾。
步驟 2：開啟遊戲
1. 電腦/筆電：直接用滑鼠雙擊 MinecraftMath.html 檔案，它會自動用瀏覽器（Chrome, Edge 等）打開。
2. 平板 (iPad/Android)：將檔案傳送到平板上，使用瀏覽器開啟即可。
步驟 3：音量控制
* 遊戲有互動音效（挖掘聲、升級聲），能增加沉浸感。
* 若在教室怕聲音太吵，畫面左上角有「靜音按鈕」可隨時關閉音效。
